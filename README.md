# 📱 Responsive Navigation Menu (CSS-Only)

> Um menu de navegação moderno, totalmente responsivo e construído exclusivamente com HTML5 e CSS3, utilizando lógica de estados nativa para máxima performance.

## 📖 Sobre o Projeto

Este projeto faz parte do meu portfólio de estudos em **Programação Full-Stack**. O objetivo principal foi desenvolver uma barra de navegação (Navbar) que se adapta perfeitamente a diferentes tamanhos de tela (desktop e mobile) sem depender de bibliotecas de terceiros ou JavaScript para sua funcionalidade principal.

## ✨ Destaques e Aprendizados Técnicos

* **Arquitetura Responsive-First:** Estrutura preparada para comportamentos distintos entre desktop (menu horizontal) e mobile (menu lateral expansível).
* **Navegação Semântica:** Implementação de tags `<nav>`, `<ul>` e `<li>` para garantir que o menu seja acessível e compreendido corretamente por motores de busca e leitores de tela.
* **Identidade Visual:** Inclusão de uma área de logo personalizada e integração com a biblioteca Font Awesome (v5.15.2) para uma interface intuitiva.
* **Interatividade Suave:** Estrutura configurada para suportar transições de movimento e efeitos de hover nos links de Início, Sobre, Serviços, Contato e Feedback.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica e lógica de formulário para controle de UI.
* **CSS3:** Estilização de layout (Flexbox/Grid), media queries para responsividade e animações.
* **Font Awesome:** Iconografia para os botões de controle do menu.
